# Architecture Decision Records (ADR)

Use this folder for significant technical decisions.

## Filename Convention

`NNN-short-kebab-title.md`

Examples:

- `001-monorepo-structure.md`
- `002-package-boundary-rules.md`

## ADR Template

```md
# NNN: Title

## Status
Accepted | Proposed | Superseded

## Context
Why this decision is needed.

## Decision
What we chose.

## Consequences
Trade-offs and follow-up actions.
```
