# Cursor Rules

Place project-specific AI guidance files in this folder.

## Purpose

- Keep coding conventions persistent and local to the repository.
- Reduce prompt repetition and avoid ambiguous implementation choices.

## Suggested Rules

- Naming and file organization standards
- Dependency/boundary constraints
- Testing expectations for changes
- Error handling and logging conventions

## Tip

Keep each rule file narrow and actionable so AI can load only relevant guidance.
